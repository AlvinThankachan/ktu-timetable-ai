# KTU Timetable AI

An AI-powered academic timetable management system built for Kerala Technological University (KTU).

## Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- AI Engine: Python (Flask + OR-Tools)
